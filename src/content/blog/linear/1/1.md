# 第一章 線形代数とは

線形代数とは何か。
線形代数とは、**線型写像**を考える学問です。

写像とは、入力を与えれば出力が定まる手続のことです。
入力が数であるものを高校でも関数と呼び、扱ってきましたね。

線型写像とは、以下の条件（線形性と言います）を満たす写像$f$のことです。
以下の$\bm{v},\bm{w}$は数とは限らないことに注意してください。
$c$は体と呼ばれる構造に含まれる数（以降、スカラー）です。
体には実数や複素数も含まれるので、よく分からなければ実数のことだと思ってください。

$$f(\bm{v}) + f(\bm{w}) = f(\bm{v} + \bm{w}) \tag{1}$$

$$cf(\bm{v}) = f(c\bm{v}) \tag{2}$$

この性質は至る所で見かけますが、特に有用なのは微分でしょう。

$$\frac{d}{dx} g(x) + \frac{d}{dx} h(x) = \frac{d}{dx} (g + h)(x)$$

$$c\frac{d}{dx} g(x)  = \frac{d}{dx} (cg)(x)$$

最初の定義において、$f=\frac{d}{dx},\bm{v} = g(x),\bm{w} = h(x)$としたものが成り立っていますね。
$\bm{v},\bm{w}$が数とは限らないというのは、このように関数を入力とした写像を考えても良いということですね。

他にも、積分や総和など、多くの有用な写像は線型写像です。
ですから、これを学ぶことは大いに価値があります。

## ベクトル空間

さて、線型写像のことについて考えたいわけですが、写像とは入力に対して働くものです。
したがって、まずは入力される対象について調べてみることにしましょう。

先ほど、当たり前のように$\bm{v} + \bm{w}$や$c\bm{v}$と言った記号を使いましたが、実はこれはルール違反です。
数でない対象に足し算やスカラー倍は一般的に定義できるとは限らないからです（みかんにりんごは足せませんね？）。
ですから、入力される対象はある程度ルールが定まったものでなければいけません。
しかし、ルールを定めすぎると適用できる議論の幅が狭まって、不便になってしまいます。
線型写像を考えるのに最低限必要なルールを考えなければいけません。

このために作り出されたものが、**ベクトル空間**$V$です。
空間というのは、集合であり、なおかつその要素同士での計算ルールが定められたものです。
ベクトル空間の要素を**ベクトル**といいます（先ほどの$\bm{v},\bm{w}$はベクトルだったのですね）。
また、ベクトルは通常太字で書かれます。
反対に、スカラーは細字で書かれます。
$(1),(2)$式を見ると、ベクトル空間には

1. ベクトル同士の足し算
1. スカラーとベクトルの掛け算

の二つが定義されている必要があることが分かりますね。
また、これらを正しく行うために、以下の8つのルールが守られる必要があります。
ただし、これは本質ではないので覚えなくて結構です。
足し算とスカラー倍が当たり前にできると言っているだけなので。

**加法に関する公理：**

- $\forall \bm{a},\bm{b},\bm{c} \in V ; (\bm{a}+\bm{b})+\bm{c} = \bm{a} + (\bm{b} + \bm{c})$　　（結合法則）
- $\forall \bm{a},\bm{b} \in V ; \bm{a} + \bm{b} = \bm{b} + \bm{a}$　　（交換法則）
- $\exists \bm{0} \in V ; \forall \bm{a} \in V, \bm{a} + \bm{0} = \bm{a}$　　（零元の存在）
- $\forall \bm{a} \in V, \exists -\bm{a} \in V ; \bm{a} + (-\bm{a}) = \bm{0}$　　（逆元の存在）

**スカラー倍に関する公理：**

- $\forall \bm{a},\bm{b} \in V, \forall c \in \mathbb{F} ; c(\bm{a}+\bm{b}) = c\bm{a} + c\bm{b}$　　（分配法則）
- $\forall \bm{a} \in V, \forall c,d \in \mathbb{F} ; (c+d)\bm{a} = c\bm{a} + d\bm{a}$　　（分配法則）
- $\forall \bm{a} \in V, \forall c,d \in \mathbb{F} ; c(d\bm{a}) = (cd)\bm{a}$　　（結合法則）
- $\forall \bm{a} \in V ; 1\bm{a} = \bm{a}$　　（単位元）

また、今までは入力側についてのみ議論していましたが、
$(1),(2)$式を見ると$f(v) + f(w)$や$cf(v)$といった記号がありますね。
これは、線型写像は出力側もベクトルでなければいけない、ということに他なりません。

まとめると、線型写像を正しく処理するための演算が整った空間をベクトル空間と言い、
線型写像は入力も出力もベクトルでなければならない、ということです。

なお、このことを線型写像$f$とベクトル空間$V,W$を用いて「$f: V \mapsto W$」と書いたりします。
覚えておくと役にたつ・・・かも。
